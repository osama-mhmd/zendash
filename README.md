# Zendash

**Debug like in a Game with Zen mode enabled**

## Introduction

Zendash is like Sentry but in Zen Mode. We hided the noise from you, so you keep focused.

## Installation

- `git clone https://github.com/osama-mhmd/zendash`
- `pnpm i`
- Fill in the env vars (found at `api/.env.example`, and `web/.env.example`)
- `pnpm run dev`
